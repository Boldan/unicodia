Что такое Юникодия?

Это программа, заменяющая Таблицу символов Windows.

  • Все символы Юникода 12, и большинство из Юникода 13/14, включая все эмодзи.
  • Простейший поиск.

What is Unicodia?

This is a replacement for Windows Character Map.

  • All characters of Unicode 12, and most of Unicode 13/14, including all emoji.
  • Simple search.
